### 二、使用

![image](Explorer.png)
<p style="color:#f60">此图展示 3.0 版本，图片仅供参考。</p>

<h5>使用步骤如下：</h5>

1. 选择所需版本<br/>
2. 下载相应版本<br/>
3. 选择核心<br/>
4. 选择存档<br/>
5. 打开选定的存档<br/>
6. 启动服务器<br/>
7. 为存档命名<br/>
8. 在Minecraft内使用"stop"命令停止服务器<br/>
9. 退出启动器<br/>
其他功能请自行探索。<br/>
